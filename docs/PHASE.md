# 개발 단계 (PHASE)

## Phase 1: UI 구성

**목표**: 기본 UI 레이아웃 구성

**내용**:
- API Key 입력 폼
- 이력서 업로드 영역
- 평가 결과 표시 영역 (레이아웃만)

**특징**: 아무 동작 안 함 (정적 UI)

**완료 조건**: 모든 UI 요소가 화면에 표시됨

---

## Phase 2: PDF 파일 로더 구현

**목표**: PDF 파일 로드 및 기본 내용 추출

**내용**:
- PDF 파일 업로드 기능 구현
- PDF 파일 파싱
- 앞 3줄만 텍스트로 추출하여 화면에 출력

**완료 조건**: PDF 업로드 후 앞 3줄이 화면에 표시됨

---

## Phase 3: PDF 파일 요약

**목표**: API를 사용한 이력서 내용 요약

**내용**:
- OpenAI API를 활용하여 PDF 내용을 요약
- 요약된 텍스트를 화면에 출력

**완료 조건**: PDF 업로드 후 요약된 내용이 화면에 표시됨

---

## Phase 4: 평가해보기

**목표**: JUDGE_SPEC에 따른 이력서 평가

**내용**:
- 5가지 평가 기준(책임감, 성격, 사교성, 성실성, 역량)에 따라 평가
- 각 기준별로 A~F 등급 결정
- 평가 결과를 간단한 텍스트로 출력
- 최종 종합 등급 출력

**완료 조건**: PDF 업로드 후 텍스트 기반 평가 결과가 화면에 표시됨

---

## Phase 5: 멋진 평가 UI로 개선

**목표**: 평가 결과를 시각적으로 매력적인 UI로 표현

**내용**:
- 평가 등급을 시각적으로 표현 (색상, 아이콘 등)
- 각 기준별 등급을 그래프나 차트로 표현
- 최종 등급을 강조하여 표시
- 사용자 경험 개선

**완료 조건**: 모든 평가 결과가 시각적으로 잘 표현됨
