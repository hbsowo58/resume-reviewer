# 이력서 분석 프로그램 요구사항 문서 (PRD)

## 개요
이력서 PDF 파일을 업로드하면 자동으로 분석하여 A~F 등급으로 평가해주는 서비스입니다.

## 목적
구직자가 자신의 이력서 품질을 객관적으로 평가받고 개선할 수 있도록 돕습니다.

- 등록된 API Key는 안전하게 저장됩니다

## 성공 기준
- 이력서가 성공적으로 업로드되고 분석됨
- 사용자가 명확한 등급과 피드백을 받음
- 모든 프로세스가 5분 이내에 완료됨
