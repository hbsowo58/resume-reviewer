# 이력서 리뷰어

이력서 PDF파일을 업로드하면 여러 기준에 의해 A ~ F 등급으로 평가해주는 앱입니다

## 기술 스택

- TypeScript
- React
- Vite

## 문서

### PRD.md (Product Requirements Document)
사용자 관점의 요구사항 문서입니다. 기술적 지식 없이 어떤 기능이 필요한지, 왜 필요한지를 설명합니다.

### JUDGE_SPEC.md (평가 기준 명세서)
이력서 평가의 기준을 정의한 문서입니다:
- 5가지 평가 기준: 책임감, 성격, 사교성/친화력, 성실성(학력/학점), 역량
- 각 기준별 A~F 등급의 구체적 기준
- 종합 평가 산정 방식

### PHASE.md (개발 단계)
프로젝트 진행을 5개 단계로 나눈 로드맵입니다:
- Phase 1: UI 구성 (정적 UI)
- Phase 2: PDF 파일 로더 (앞 3줄 출력)
- Phase 3: PDF 요약 (API 사용)
- Phase 4: 평가 기능 (텍스트 출력)
- Phase 5: 평가 UI 개선 (시각화)

## Harness

- 사용자가 명시적으로 요청한 것만 수행합니다
- 사용자가 지시하지 않은 추가 기능, 리팩토링, 최적화 등은 하지 않습니다
- 사용자의 지시가 모호한 경우 확대 해석하지 말고, 확실해질 때까지 사용자에게 다시 물어봅니다
